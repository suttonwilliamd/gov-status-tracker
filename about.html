About Gov Status Tracker

A real-time monitoring tool for U.S. government digital services

What's This Project?

My name is William. I created this website to provide a simple way to check if important government websites are operational. The project began as a quick experiment but has evolved into a tool that many people find useful during government shutdowns and service interruptions.

How It Works

The service checks each domain with a simple HEAD request, respecting rate limits and caching responses appropriately. All checks are performed server-side to ensure reliability and protection of user privacy.

This project was built with vanilla JavaScript and is hosted on Vercel.
